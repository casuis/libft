## ft_atoi
***

### Idée global
Convertion d'une string en integer.

### Prototype
```
int   ft_atoi(char *str)
```
### Arguments
Une chaine de caracteres remplis de chiffres.

### Return
Retourne un int, retourne 0 si rien n'est a convertir.

#### Appel
```
char  *str;
int   ret

str = "42";
ret = ft_atoi(str);
```
